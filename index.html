What is CI/CD?

CI/CD is a method used in cloud computing to automate building and deploying applications. Continuous Integration means automatically testing and checking code when developers push updates. Continuous Deployment means automatically publishing the updated application. This helps developers release updates faster and reduces manual work.

My Pipeline

When I push code to GitHub, GitHub Actions automatically runs a workflow. It checks if the required files exist, then prepares the website files. Finally, the workflow deploys the website to GitHub Pages so it becomes available online.
